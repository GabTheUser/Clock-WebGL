# Часы синхронизации с сервером на WebGL

## Описание проекта

Это проект на Unity, в котором реализованы часы, синхронизируемые с сервером времени через API Яндекса. Часы отображают текущее время с анимацией стрелок (с помощью библиотеки **DoTween**) и предоставляют возможность ручного редактирования времени через пользовательский интерфейс (UI). Проект предназначен для сборки на платформу **WebGL**.

## Функциональность

- **Синхронизация времени с сервером**: Приложение получает текущее время с сервера Яндекс через API.
- **Анимация стрелок**: Для плавного движения стрелок используется библиотека **DoTween**.
- **Ручное редактирование времени**: Пользователь может вручную изменять время через поля ввода (часы, минуты, секунды) или перетаскивать стрелки.
- **Визуальные подсказки**: Включен режим редактирования, отображается подсказка для пользователя.

## Скрипты

### 1. `ClockController.cs`

Этот скрипт отвечает за получение текущего времени с сервера и обновление его каждые 60 минут.

**Основные функции:**
- Получение времени с сервера Яндекса через **UnityWebRequest**.
- Передача времени в контроллер стрелок (скрипт `ClockHandControl`).

### 2. `ClockHandControl.cs`

Этот скрипт управляет положением стрелок часов, минут и секунд, а также полями ввода времени.

**Основные функции:**
- Установка времени и обновление положения стрелок на основе полученного времени.
- Анимация движения стрелок с использованием **DoTween**.
- Поддержка ручного редактирования времени через поля ввода.

### 3. `TimeEditer.cs`

Этот скрипт управляет ручным режимом редактирования стрелок.

**Основные функции:**
- Включение/выключение режима редактирования.
- Перетаскивание стрелок для изменения времени.
- Синхронизация введенного времени через **InputField** с анимацией стрелок.


## Используемые технологии

- **Unity 2022.3.47**
- **WebGL** для публикации проекта в браузере
- **DoTween** — для анимации вращения стрелок
- **UnityWebRequest** — для получения времени с сервера
- **API Яндекса** — для получения точного времени

## API для времени

Проект использует API для синхронизации времени: https://yandex.com/time/sync.json

Ответ от API содержит текущее время в виде Unix timestamp, которое преобразуется в локальное время с помощью **DateTimeOffset**.

## Как редактировать время

1. Нажмите кнопку "Edit Time", чтобы включить режим редактирования.
2. Либо введите время через поля (часы, минуты, секунды), либо перетащите стрелки на нужные позиции.
3. Нажмите кнопку "Confirm", чтобы сохранить новое время, и стрелки начнут движение с новой точки.
